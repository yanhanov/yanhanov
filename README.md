There was a time I used to gaze at websites and wonder, *"Someone out there builds this."*  
**Now — I am that someone.**

I never went to college, never paid for expensive courses, and I wear no titles I didn't earn.  
One day, I simply opened an HTML file and began.

Since then, each line of code feels like a line in my own book — written carefully, with meaning, and above all — **so it works**.
